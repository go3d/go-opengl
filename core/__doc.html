A cross-platform Go binding for OpenGL core profile 3.3 and/or higher with extensions EXT_texture_filter_anisotropic, EXT_texture_compression_s3tc, EXT_texture_sRGB. Auto-generated using github.com/go3d/go-opengl/cmd/gen-opengl-bindings

To initialize this OpenGL binding, after having created your GL context (with SDL, GLFW or in some other way) call glcore.Util.Init().
